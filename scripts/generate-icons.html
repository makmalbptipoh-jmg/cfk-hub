Langkah Selepas Muat Turun:
  1. Muat turun kedua-dua fail PNG (icon-192.png dan icon-512.png)
  2. Salin ke folder cfk-hub/public/
  3. Jalankan npm run build untuk rebuild
  4. Deploy ke Vercel — PWA kini bersedia dipasang!
