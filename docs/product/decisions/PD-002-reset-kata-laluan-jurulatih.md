# PD-002: Bagaimana Admin menetapkan semula kata laluan Jurulatih?

## Status
Diputuskan

## Soalan
Apakah aliran produk untuk menetapkan semula kata laluan akaun Jurulatih yang terlupa atau perlu ditukar?

## Konteks
Sistem adalah dalaman sahaja tanpa pendaftaran diri. Jika Jurulatih terlupa kata laluan, Admin perlu ada cara untuk membantu tanpa memerlukan sistem emel yang kompleks.

## Pilihan
- **A** — Admin tetapkan semula kata laluan terus dari panel pengurusan pengguna (Admin taip kata laluan baharu untuk Jurulatih)
- **B** — Sistem hantar pautan reset ke emel Jurulatih secara automatik
- **C** — Admin tetapkan kata laluan sementara, Jurulatih diwajibkan tukar semasa log masuk pertama

## Keputusan
**Pilihan A** — Admin tetapkan semula kata laluan terus dari panel pengurusan pengguna. Admin taip kata laluan baharu untuk akaun Jurulatih berkenaan, kemudian maklumkan kata laluan baharu kepada Jurulatih secara lisan atau melalui WhatsApp.

## Sebab
Sistem CFK HUB adalah sistem dalaman yang hanya digunakan oleh 1 Admin dan 6 Jurulatih. Memerlukan sistem e-mel automatik (Pilihan B) menambah kerumitan teknikal yang tidak perlu. Pilihan A lebih mudah, pantas, dan sesuai dengan persekitaran pasukan kecil di mana Admin dan Jurulatih saling kenal dan mudah dihubungi.

## Kesan
- Panel pengurusan pengguna Admin perlu ada butang "Tetapkan Semula Kata Laluan" untuk setiap akaun Jurulatih
- Admin masukkan kata laluan baharu secara terus (tanpa e-mel/OTP)
- Admin bertanggungjawab memaklumkan kata laluan baharu kepada Jurulatih berkenaan
- Tiada aliran "lupa kata laluan" untuk Jurulatih — semua melalui Admin
