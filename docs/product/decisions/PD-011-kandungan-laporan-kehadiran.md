# PD-011: Apakah maklumat yang dipaparkan dalam laporan kehadiran untuk ibu bapa?

## Status
Diputuskan

## Soalan
Apakah kandungan dan format laporan kehadiran PDF yang akan dikongsi kepada ibu bapa?

## Konteks
Laporan kehadiran untuk ibu bapa perlu memberikan maklumat yang bermakna tanpa mendedahkan maklumat pelajar lain atau data dalaman CFK. Reka bentuk laporan mempengaruhi kepercayaan ibu bapa terhadap program.

## Pilihan
- **A** — Ringkas: Nama pelajar, bulan, jumlah hadir, jumlah tidak hadir, peratusan kehadiran
- **B** — Terperinci: Nama pelajar, bulan, senarai tarikh setiap sesi (Hadir/Tidak Hadir), jumlah dan peratusan
- **C** — Penuh: Semua maklumat Pilihan B + komen Jurulatih (jika ada) + logo CFK + tandatangan Admin

## Keputusan
**Pilihan B** — Laporan kehadiran standard dengan maklumat berikut:

| Bahagian | Kandungan |
|---|---|
| Kepala laporan | Logo CFK, nama cawangan, tajuk "Laporan Kehadiran" |
| Maklumat pelajar | Nama penuh pelajar, cawangan daftar |
| Tempoh | Bulan dan tahun laporan |
| Jadual kehadiran | Senarai tarikh setiap sesi + status: Hadir / Tidak Hadir / Cuti |
| Ringkasan | Jumlah sesi, bilangan hadir, bilangan tidak hadir, peratusan kehadiran |
| Nota | Nota cuti (jika ada, dari rekod Jurulatih) |

## Sebab
Senarai tarikh per sesi membenarkan ibu bapa semak hari-hari spesifik anak mereka tidak hadir — lebih bermakna daripada sekadar angka peratusan. Tidak terlalu kompleks untuk dibangun dan cukup profesional untuk dikongsi.

## Kesan
- Templat PDF laporan kehadiran perlu direka bentuk dengan jadual per sesi
- Sistem perlu tarik semua rekod kehadiran pelajar untuk bulan yang dipilih
- Status "Cuti" dalam laporan perlu tunjuk nota ringkas jika ada (dari PD-005)
- Admin pilih pelajar + bulan → sistem jana PDF → Admin kongsi kepada ibu bapa (melalui WhatsApp atau lain)
- Cawangan sesi (bukan cawangan daftar) dipaparkan dalam setiap baris jadual jika pelajar hadir ke cawangan lain
