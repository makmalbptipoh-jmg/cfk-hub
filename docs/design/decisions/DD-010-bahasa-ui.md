# DD-010: Apakah bahasa antara muka (UI) sistem?

## Status
Diputuskan

## Soalan
Apakah bahasa yang digunakan untuk label, butang, mesej, dan semua teks dalam antara muka CFK HUB?

## Keputusan
**Bahasa Melayu sahaja** — semua elemen UI dalam Bahasa Melayu.

### Panduan Istilah Standard

| Elemen | Teks BM |
|---|---|
| Butang simpan | Simpan |
| Butang batal | Batal |
| Butang tambah | + Tambah / Tambah Baharu |
| Butang edit | Edit |
| Butang padam/nyahaktif | Nyahaktifkan |
| Butang sahkan | Sahkan |
| Butang kembali | ← Kembali |
| Carian | Cari... |
| Tiada rekod | Tiada rekod dijumpai |
| Muatkan semula | Muatkan Semula |
| Log keluar | Log Keluar |
| Status Hadir | Hadir |
| Status Tidak Hadir | Tidak Hadir |
| Status Cuti | Cuti |
| Belum bayar | Belum Bayar |
| Sudah bayar | Sudah Bayar |
| Aktif | Aktif |
| Tidak Aktif | Tidak Aktif |
| Dibatalkan | Dibatalkan |

### Mesej Sistem

| Situasi | Mesej |
|---|---|
| Berjaya simpan | "Rekod berjaya disimpan." |
| Ralat | "Ralat berlaku. Sila cuba semula." |
| Sahkan padam | "Adakah anda pasti? Tindakan ini tidak boleh dibatalkan." |
| Tiada data | "Tiada rekod untuk dipaparkan." |
| Medan wajib | "Sila isi medan ini." |
| Import berjaya | "X rekod berjaya diimport." |

## Sebab
Semua pengguna CFK HUB (1 Admin dan 6 Jurulatih) adalah rakyat Malaysia yang fasih berbahasa Melayu. Sistem dalaman tidak memerlukan dwibahasa. Bahasa Melayu yang konsisten meningkatkan keselesaan pengguna dan mengurangkan kekeliruan istilah. Selaras dengan laporan kehadiran ibu bapa yang juga dalam BM (PD-010).

## Kesan
- Semua label, placeholder, tooltip, mesej ralat, dan mesej berjaya dalam BM
- Nombor dan tarikh mengikut format Malaysia: DD/MM/YYYY, RM, bulan dalam BM (Januari, Februari...)
- Nama bulan dalam laporan: "Januari 2026", "Februari 2026"
