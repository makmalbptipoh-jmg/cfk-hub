# DD-002: Apakah palet warna utama sistem?

## Status
Diputuskan

## Soalan
Apakah warna-warna utama yang membentuk identiti visual CFK HUB?

## Keputusan
Palet tiga warna: **Charcoal + Lime + Soft Gray**

### Warna Utama

| Peranan | Nama | Hex | Kegunaan |
|---|---|---|---|
| **Primer** | Charcoal | `#1E293B` | Sidebar, header, teks utama, butang primer |
| **Aksen** | Lime | `#84CC16` | Butang CTA, elemen aktif, badge, highlight penting |
| **Latar** | Soft Gray | `#F1F5F9` | Latar halaman, kad, kawasan kandungan |

### Warna Sokongan

| Peranan | Nama | Hex | Kegunaan |
|---|---|---|---|
| Teks Utama | Dark Slate | `#0F172A` | Teks dalam kandungan |
| Teks Sekunder | Cool Gray | `#64748B` | Label, metadata, placeholder |
| Sempadan | Border Gray | `#E2E8F0` | Garis pemisah, sempadan kad |
| Putih | White | `#FFFFFF` | Latar kad, modal, input |
| Lime Gelap | Lime 700 | `#4D7C0F` | Hover state butang lime |
| Charcoal Cerah | Slate 700 | `#334155` | Hover state sidebar item |

### Contoh Penggunaan

```
Sidebar:          Latar #1E293B, Teks #F1F5F9, Aktif #84CC16
Header/Topbar:    Latar #FFFFFF, Sempadan #E2E8F0
Kandungan:        Latar #F1F5F9
Kad:              Latar #FFFFFF, Sempadan #E2E8F0
Butang Primer:    Latar #1E293B, Teks #FFFFFF
Butang Aksen:     Latar #84CC16, Teks #1E293B
```

## Sebab
Charcoal memberikan kesan profesional dan berwibawa yang sesuai untuk sistem pengurusan. Lime sebagai aksen memberikan tenaga dan moden tanpa terlalu playful. Soft Gray sebagai latar mengurangkan keletihan mata semasa penggunaan berjam-jam.

## Kesan
- Sidebar Admin menggunakan latar Charcoal dengan item aktif berwarna Lime
- Semua butang utama (Simpan, Jana, Sahkan) menggunakan Lime atau Charcoal
- Latar halaman kekal Soft Gray untuk membezakan dengan kad putih
