# DR-008: Adakah resit dikeluarkan dalam format digital atau cetak?

## Status
Diputuskan

## Soalan
Apakah format pengeluaran resit yang digunakan oleh sistem CFK HUB?

## Konteks
Format resit menentukan teknologi yang digunakan (PDF generator, printer integration) dan pengalaman pengguna dalam menerima resit.

## Keputusan
Resit dikeluarkan dalam format **digital (PDF) yang juga boleh dicetak**. PDF boleh dimuat turun, dikongsi melalui WhatsApp/e-mel, dan dicetak menggunakan printer standard jika diperlukan.

## Sebab
Mengurangkan kos percetakan dengan lalai kepada PDF digital, tetapi memberi fleksibel kepada Admin untuk cetak jika ibu bapa memerlukan resit fizikal. Kos tambahan hampir sifar jika layout PDF direka bentuk untuk A4.

## Kesan
- Sistem perlu menjana fail PDF untuk setiap resit
- Layout PDF perlu sesuai untuk paparan skrin DAN cetakan A4
- Resit PDF perlu boleh dimuat turun dan dikongsi dari dalam sistem
- Tiada integrasi printer khas diperlukan — Admin guna printer sedia ada
